Descripción del curso

Este curso cubrirá diferentes modelos económicos que tratan de responder, entre otras, las siguientes preguntas: ¿Cuáles son las causas de las expansiones y de las recesiones económicas? ¿Qué factores determinan el nivel del producto, la inﬂación y el desempleo? ¿Qué políticas económicas puede utilizar el gobierno para ayudar a estabilizar el producto agregado o el nivel de precios?
